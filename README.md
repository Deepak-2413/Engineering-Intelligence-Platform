\# Engineering Intelligence Platform (EIP)



An AI-powered platform that analyzes software repositories, builds code dependency knowledge graphs, performs architecture analysis, and provides engineering insights through graph intelligence and AI-assisted querying.



\## Planned Features



\- Repository Analysis

\- Knowledge Graph Generation

\- Architecture Visualization

\- Impact Analysis

\- AI Engineering Assistant

\- Repository Health Scoring



\## Tech Stack



\- Python

\- FastAPI

\- Neo4j

\- Gemini API

\- React

\- Docker
